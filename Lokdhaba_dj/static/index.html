Get started

    API Endpoint

      http://lokdhaba.ashoka.edu.in/apiDj/v1/

The LokDhaba API provides programmatic access to the LokDhaba-website. Directly download data, while specifying the parameters you need. The API also provides easy access to the rich visualizations served by the website.

The API can be accessed by anyone without an API-key.

Download Data



# Here is a curl example
curl -X GET \
  'http://lokdhaba.ashoka.edu.in/apiDj/v1/DirectDownload/?ElectionType=GE&StateName=Delhi&AssemblyNo=3' \
  -H 'cache-control: no-cache' \
  -H 'content-type: application/json' \


#Here is a python3 example
    import requests

    url = "http://lokdhaba.ashoka.edu.in/apiDj/v1/DirectDownload/"
    querystring = {"ElectionType":"GE","StateName":"Delhi","AssemblyNo":"3"}
    headers = {
        'content-type': "application/json",
        'cache-control': "no-cache",
        'postman-token': "a75d7004-d35d-d9d7-17ae-9cf9e65fc6ac"
        }

    response = requests.request("GET", url, headers=headers, params=querystring)
    print(response.text)

DATA DOWNLOAD (POST) 



# Here is a curl example
curl -X POST \
  https://lokdhaba.ashoka.edu.in/apiDj/v1/DataDownload/ \
  -H 'cache-control: no-cache' \
  -H 'content-type: application/json' \
  -H 'postman-token: 9a7c3150-6e19-987b-eada-b59dbced38e0' \
  -d '{"ElectionType":"AE","StateName":"Delhi","AssemblyNo":"3","Filters":[{"id":"Sex","value":"M"}],"SortOptions":[]}'

#Here is a python3 example
import requests

url = "https://lokdhaba.ashoka.edu.in/apiDj/v1/DataDownload/"

payload = "{\"ElectionType\":\"AE\",\"StateName\":\"Delhi\",\"AssemblyNo\":\"3\",\"Filters\":[{\"id\":\"Sex\",\"value\":\"M\"}],\"SortOptions\":[]}"
headers = {
    'content-type': "application/json",
    'cache-control': "no-cache",
    'postman-token': "c8a4b80b-b7ab-71c4-66d3-ec2dc182731d"
    }

response = requests.request("POST", url, data=payload, headers=headers)

print(response.text)

Get Visualization Data



# Here is a curl example
curl -X POST \
http://lokdhaba.ashoka.edu.in/apiDj/v1/getVizData/ \
  -H 'cache-control: no-cache' \
  -H 'content-type: application/json' \
  -d '{"ElectionType":"GE","StateName":"Delhi","ModuleName":"winnerGenderMap","VizType":"Map","AssemblyNo":14,"Party":"","Legends":["Male","Female","Others"]}'

Glossary

The glossary houses extra information on API-functions which might come handy when experimenting with the required parameters of an API call.

MANDATORY PARAMETERS FOR DOWNLOAD DATA:

None.
This functions checks for the supplied parameters and returns a response accordingly. If one leaves a specific field blank, for example 'Election_Type', then the response contains all election data, with respect to other parameters provided.

MANDATORY PARAMETERS FOR GET VISUALIZATION DATA:

To access visualization data one needs to supply the requisite paramaters in the POST request to get a successful response. The parameters needed for the getting viz data, are listed below. These params are necessary because each visualization is mapped to a specific set of parameters.

Field Type Description Datatype
ElectionType Text The election type of the query(s) String
StateName Text The state name of the query(s) String
AssemblyNo Number Assembly number of the query(s). String
ModuleName Text Name of visualisation module String
VizType Text Type of visualization String
Party Text The party names which are required String
Filters Number Filter type of the query(s). String

Errors

The Lokdhaba API uses the following error codes:

Error Code Meaning
X000 Some parameters are missing. This error appears when you don't pass every mandatory parameters.
